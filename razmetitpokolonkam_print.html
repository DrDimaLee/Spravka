1.1.1.20. РазметитьПоКолонкам

Процедура РазметитьПоКолонкам(ТаблицаЗначений)
     Счетчик = 0;
     ПредыдущийПериодКонтроля = Справочники.ПериодыКонтроля.ПустаяСсылка();
     Для Каждого Строка Из ТаблицаЗначений Цикл
          Если Строка.ПериодКонтроля <> ПредыдущийПериодКонтроля Тогда
               Счетчик = Счетчик + 1;
          КонецЕсли;
          Строка.НомерКолонки = Счетчик;
          ПредыдущийПериодКонтроля = Строка.ПериодКонтроля;
     КонецЦикла;    
КонецПроцедуры
